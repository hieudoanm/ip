## 📦 Installation

```bash
curl -fsSL https://github.com/hieudoanm/ip/raw/refs/heads/master/packages/cli/bin/ip -o ip
chmod +x ip
sudo mv ip /usr/local/bin/
```

## 🚀 Usage

```bash
ip
ip dns <domain>
```
